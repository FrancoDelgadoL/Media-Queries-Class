Responsive web design

Es un enfoque de diseño que nos permite que nuestras paginas web se visualicen de forma correcta en varias dispositivos con diferentes tamaños de pantalla.

El viewport es el area visible del navegador y es importante establecer el viewport para controlar las dimensiones de la pagina.

Los Media Queries nos permiten aplicar estilos CSS diferentes dependiendo del tamaño del dispositivos donde estemos visualizando la pagina web.
